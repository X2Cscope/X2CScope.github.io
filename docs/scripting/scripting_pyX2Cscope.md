# Quick start

```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate
# Install pyX2Cscope
pip install pyx2cscope
# Start demo GUI
python -m pyx2cscope
```

Further documentation of pyX2Cscope usage:
https://x2cscope.github.io/pyx2cscope/
